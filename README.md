# Corona Case Viewer
This is a desktop based application which extract data from a webstite <a href='https://coronaclusters.in/'>Corona Cluster</a>

## Used Modules

bs4 (extrating data)

urllib (getting website request)

tkinter (GUI of the application)

## Instruction to use

Open GUI.py
